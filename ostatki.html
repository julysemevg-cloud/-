Telegram Bot сохранено в браузере
Токен бота не хранится в коде файла — вводится здесь и сохраняется только в локальной памяти этого браузера (localStorage). Никому не передавайте этот файл с заполненными полями и не публикуйте его в репозитории.
Если токен где-то "засветился" — перевыпустите его через @BotFather: /mybots → Bot Settings → API Token → Revoke current token.
Как получить Chat ID

Для личного чата или группы — перешли любое сообщение боту @getmyid_bot.
Для канала или супергруппы ID начинается с -100...
Thread ID — это номер топика внутри группы-форума.
